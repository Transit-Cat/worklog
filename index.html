One-time setup. You only do this once per device.

1
Create a free Firebase project

Go to console.firebase.google.com and sign in. Click Add project, give it any name, disable Analytics if asked, click Create project.

2
Register a web app

Inside your project click the </> (Web) icon. Give it any nickname, click Register app. Keep the firebaseConfig block open.

3
Enable Realtime Database

Left sidebar: Build → Realtime Database → Create Database. Choose any region, select Start in test mode, click Enable.

Your database URL looks like: https://your-project-default-rtdb.firebaseio.com

4
Paste your config values
